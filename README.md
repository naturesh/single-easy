# What is Single easy

Single easy란 영어문장 암기를 위한 싱글 웹페이지로 Svelte, Tailwindcss 로 제작되었습니다.

## Why?

학교 생활중 영어문장 암기가 필요한 순간이 있었고 이러한 때에 이런 암기 프로그램이 있으면 좋겠다 생각해서 만들게 되었습니다.

## Developing

개발 과정중 큰 어려움은 없었으나 Sveltekit을 github actions을 통해 배포하는데 막히는 부분이 있었습니다.

## How to use

 - L4
 - L5
 - RE : n

 이렇게 3개의 버튼이 있는데

 ```
 L4 : lesson4
 L5 : lesson5
 RE : N : 복습, 오답노트 개수
```
